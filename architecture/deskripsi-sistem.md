# Deskripsi Sistem

## Nama Sistem

Sistem Manajemen Laundry

## Tujuan

Sistem ini dibuat untuk membantu pengelolaan data pelanggan, transaksi laundry, status cucian, dan laporan sehingga proses pelayanan menjadi lebih cepat, akurat, dan efisien.

## Pengguna

- Admin
- Kasir
- Pelanggan

## Fitur Utama

- Login
- Data Pelanggan
- Penerimaan Laundry
- Status Proses Laundry
- Pembayaran
- Laporan Transaksi
