# Quality Attributes

## Performance

Sistem mampu memproses transaksi laundry dengan cepat sehingga pelayanan menjadi lebih efisien.

## Security

Data pelanggan dan transaksi dilindungi menggunakan autentikasi pengguna dan enkripsi password.

## Usability

Antarmuka dibuat sederhana sehingga mudah digunakan oleh admin maupun kasir.

## Reliability

Sistem mampu beroperasi dengan stabil tanpa sering mengalami gangguan.

## Scalability

Sistem dapat dikembangkan untuk mendukung banyak cabang usaha laundry.
