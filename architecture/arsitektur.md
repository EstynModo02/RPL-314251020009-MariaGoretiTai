# Arsitektur Sistem

## Pola Arsitektur

Layered Architecture

## Lapisan Sistem

### Presentation Layer

Menampilkan antarmuka aplikasi kepada pengguna.

### Business Layer

Mengelola proses login, penerimaan laundry, pencatatan status cucian, pembayaran, dan pembuatan laporan.

### Data Access Layer

Menghubungkan aplikasi dengan database.

### Database Layer

Menyimpan seluruh data pelanggan, transaksi, status cucian, dan laporan.

## Alasan Pemilihan

Layered Architecture dipilih karena mudah dipelihara, mudah dikembangkan, dan memisahkan fungsi setiap bagian sistem.
