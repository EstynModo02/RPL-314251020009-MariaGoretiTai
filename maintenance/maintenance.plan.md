# Maintenance Plan

## Corrective

1. Memperbaiki bug login yang gagal meskipun password benar.
2. Memperbaiki kesalahan validasi data pada form pendaftaran.
3. Memperbaiki error saat pengguna mengubah profil.

## Adaptive

1. Menyesuaikan aplikasi dengan versi Android terbaru.
2. Mengubah API pembayaran ke versi terbaru.
3. Menyesuaikan sistem dengan perubahan kebijakan pemerintah.

## Perfective

1. Menambahkan fitur pencarian produk.
2. Meningkatkan kecepatan loading halaman.
3. Menambahkan fitur notifikasi kepada pengguna.

## Preventive

1. Melakukan backup database secara berkala.
2. Melakukan refactoring kode agar lebih mudah dipelihara.
3. Memperbarui dokumentasi sistem secara rutin.
